How it Works

StudyWidgets works with your notes. You create decks of cards by typing your notes, copying and then pasting into the new deck form.

You must use a format that StudyWidgets understands. The easiest is a format that works well with Microsoft OneNote (but can also be used in a text editor of your choice). Both Windows Store and Web versions of OneNote are free.

  1. Type your title, press enter
  2. Type the first term, press tab
  3. Type the definition, press enter

The results in OneNote create a table for you automatically. You can create multiple decks simultaneously, just separate decks by pressing enter.
